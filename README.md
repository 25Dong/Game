## 环境搭建
· VSCode取消导入.meta文件（该文件记录Cocos编辑器修改内容，VS编程代码的时候并不关心这类型的文件）
设置——>File exclude （**/*.meta）

## 《像素鸟》
### 知识点
- 背景滚动
- 物理碰撞（代码中自动开启碰撞引擎，禁止旋转，静态刚体和拥有重力的刚体，修改速度）
- 动画（帧动画，加载播放，循环播放）
- TS语法（脚本引入）
